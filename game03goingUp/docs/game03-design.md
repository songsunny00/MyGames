# 游戏设计文档：爬楼 (Going Up)

> **版本：** v1.1
> **日期：** 2026-04-03
> **状态：** 开发中

---

## 一、概念与愿景

**一句话定位：** 一款以"打工人逃离职场"为情感核心的跑酷答题游戏。玩家控制角色沿Z型楼梯自动向上爬，每隔 10-15 层遇到一个障碍，答题解锁，答对"去班味"答案继续爬，最终逃到 365 层顶楼，直升机接走撒祝福。

**核心情感：** 打工人不是在"玩游戏"，而是在游戏里完成一次"精神离职"。每一道答对的题都是一次爽感输出——正确答案就是那个你不敢说出口的心里话。

**目标用户：** 25-40 岁职场人群，微信生态碎片化消费，重度手机用户，有职场共鸣。

---

## 二、设计支柱

> 所有设计决策必须对照以下支柱进行衡量。

1. **去班味爽感** — 正确答案就是"摆烂"，让玩家在游戏里说出平时不敢说的话
2. **轻量即开即玩** — 单局 3-5 分钟，适合碎片时间，不需要理解复杂系统
3. **情感弧线完整** — 从"被困职场"到"直升机撒祝福"，有起承转合的体验
4. **截图传播友好** — 终局祝福语、楼层成就，天然适配朋友圈传播

## 三、核心游戏循环

```
[角色自动沿Z型楼梯向上爬]
    ↓
[左右滑动控制水平移动 / 点击跳跃躲避缺口]
    ↓ 碰到 ❓ 方块
[进入答题模式] ← 屏幕暂停，弹出问答
    ├─ 答对 → 继续爬楼，楼层 +1，庆祝动效
    └─ 答错 → 随机触发惩罚 A 或 D
              ├─ A: 掉落 3-5 层（屏幕回滚）
              └─ D: 进入"班味惩罚"状态（视觉变形，屏幕中央显示嘲讽文字，需再答一题解除）
    ↓ 碰到 💣 炸弹（从上方掉落）
[炸弹惩罚] ← 屏幕暂停，随机触发 A 或 B
    ├─ A: 直接从头再玩（显示"被炸飞了！"）
    └─ B: 弹出分享免死弹窗
          ├─ 分享成功 → 继续游戏
          └─ 重新开始 → 从第 1 层重新开始
    ↓ 碰到 🎁 礼包（放置在楼梯上）
[礼包奖励] ← 立即触发，无需操作
    ├─ A: 鼓励特效（文字 + 粒子）
    └─ B: 跳跃 +5/+10 层
    ↓ 到达第 365 层
[终局动画] → 直升机起飞 + 撒大礼包祝福 + 分享按钮
```

### 3.1 即时循环（0-10秒）

- 角色自动沿Z型楼梯向上移动
- 玩家左右滑动控制水平位置，躲避炸弹
- 点击/上滑跳跃越过断裂缺口
- 节奏轻快，操作零学习成本

### 3.2 会话循环（3-5分钟）

- 每 10-15 层遇到一个 ❓ 方块
- 答题模式：4 选 1（或 3 选 1），有紧迫感但不焦虑
- 惩罚带来张力，但不致命——失败可以重来
- 随机掉落炸弹（动态概率）和礼包（放置在楼梯上），增加随机性和爽感

### 3.3 长期循环

- 通关 365 层 = 达成"365天逃离班"成就
- 题库足够支撑多次重玩不重复
- 分享终局截图触发社交传播
- **收集系统**：三轮收集机制，每次通关后依次收集一个，最多3次通关即可收集完毕
  - **第一轮**：摸鱼徽章（如"准时下班"、"摸鱼大师"、"绩效摆烂王"、"365天零加班"等）
  - **第二轮**：离职宣言（如"老子不干了！"、"感谢公司培养，但我更感谢自己的勇气"、"KPI永远差一点，人已到海南了"等）
  - **第三轮**：旅游目的地（大理、三亚、日本、欧洲等，终局动画背景随之变化）
- **收集展示**：游戏界面右上角有收集入口，点击弹出弹窗展示三格（徽章/宣言/目的地），未解锁的格子显示问号
- **实时进度展示**：收集入口旁显示进度指示器（如"1/3"、"2/3"、"3/3"），激活玩家攀比心理
- **收集完成奖励**：收集完毕后触发烟花特效，送上一个祝福，如"恭喜你彻底逃离班味！"

## 四、核心玩法详细设计

### 4.1 移动与跳跃（参考神庙逃亡）

| 参数         | 值                                                                 |
| ------------ | ------------------------------------------------------------------ |
| 移动方式     | 角色自动沿Z型楼梯向上移动                                          |
| 水平控制     | 左滑向左移动 / 右滑向右移动                                        |
| 跳跃方式     | 点击或上滑跳跃（点按=小跳，长按=大跳）                             |
| 移动边界     | 角色不能超出楼梯边缘                                               |
| 楼梯结构     | Z型连续楼梯，奇数层从右往左，偶数层从左往右                        |
| **难度变化** | 第二次通关时楼梯速度增加，第三次通关时速度进一步增加，之后封顶     |

**跳跃机制：**
- **小跳**：点按或短按上滑，跳跃高度较低，适合小缺口
- **大跳**：长按后上滑，跳跃高度较高，适合大缺口
- 长按判定时间：0.15秒

**难度封顶机制：**
- 第三次通关后，楼梯速度不再增加，保证游戏可通关性
- 高难度关卡增加道具掉落率（礼包概率提升至 20%），平衡游戏体验
- 第1关楼梯连续无缺口，第2关起出现断裂缺口

### 4.2 楼梯系统

| 参数     | 值                                             |
| -------- | ---------------------------------------------- |
| 楼梯结构 | Z型连续楼梯，每层一个斜坡段                    |
| 楼层方向 | 奇数层从右往左，偶数层从左往右                 |
| 楼梯宽度 | 初始宽度较宽，随层数增加略微缩窄               |
| 断裂缺口 | 第2关起随机出现，需要跳跃越过                  |

**楼梯渲染：**
- 每层为一个斜坡段，带圆角和高光效果
- 楼梯颜色随楼层渐变（黄绿色系）
- 拐角处有圆形转角标记

### 4.3 障碍系统

| 参数     | 值                                             |
| -------- | ---------------------------------------------- |
| 障碍频率 | 每 10-15 层随机出现一个 ❓ 方块                |
| 障碍外观 | 红色问号方块，带轻微浮动动画                   |
| 障碍位置 | 放置在楼梯上                                   |
| 碰撞触发 | 角色碰到 ❓ 方块 → 立即暂停游戏 → 弹出答题面板 |

### 4.4 答题系统

**触发时机：** 角色碰撞 ❓ 方块时

**面板内容：**

- 题目：脑筋急转弯 / 职场反卷问答（中文）
- 选项：2-4 个选项（根据题目类型调整）
- 正确选项：符合"去班味、反卷"价值观的答案

**反馈动画：**

- 答对：选项变绿 + "去班味成功！"文字 + 继续游戏
- 答错：选项变红 + 正确答案高亮 + 随机惩罚触发

**题库规模：** 100 道（支持多轮游戏不重复抽取）

分类：

- 脑筋急转弯类（约 40%）
- 职场反卷问答类（约 60%）

**难度变化：**
- 第二次通关：题目从题库中不重复抽取，楼梯速度增加
- 第三次通关：题目继续不重复抽取，楼梯速度进一步增加

示例题目：

| #   | 类型       | 题目                                   | 正确答案                              |
| --- | ---------- | -------------------------------------- | ------------------------------------- |
| 1   | 反卷       | 周一早上被领导问"周末方便开会吗"，你： | B. 装死不回，周二再回"不好意思没看到" |
| 2   | 脑筋急转弯 | 什么东西越洗越脏？                     | 水                                    |
| 3   | 反卷       | 领导在群里发"收到请回复"，你：         | B. 等10个人回复后，第11个回"好的"     |
| 4   | 脑筋急转弯 | 什么瓜不能吃？                         | 傻瓜                                  |

### 4.5 惩罚系统

**触发条件：** 答题答错时，随机选择 A 或 D

| 类型                 | 效果                                 | 视觉表现                                                                                      |
| -------------------- | ------------------------------------ | --------------------------------------------------------------------------------------------- |
| **惩罚 A：掉楼**     | 楼层回退 3-5 层（分段惩罚）          | 角色快速下落动画，屏幕回滚，显示安慰文字"差一点就过了！"                                                                    |
| **惩罚 D：班味惩罚** | 角色进入"班味"状态，需再答对一题解除 | 角色变成西装领带外观 + 头顶"加班中"标签，屏幕中央显示嘲讽文字"你的班味太重了！"，视觉被"污染" |

**分段惩罚机制：**
- 楼层 < 100：掉落 3 层
- 楼层 100-250：掉落 4 层
- 楼层 > 250：掉落 5 层

**掉楼安慰文字池（随机一条）：**
- 差一点就过了！
- 别灰心，继续加油！
- 下次一定行！
- 摸鱼失败，再试一次！
- 离成功只差一点点！

**班味惩罚嘲讽文字池（随机一条）：**

- 你的班味太重了！
- 又被卷进去了？
- 打工魂觉醒中...
- 你的发际线在抗议！
- 加班快乐（并不）
- 老板在看着你！
- PPT写完了吗？
- 你的 KPI 还差多少？

- 班味惩罚状态：角色外观变为"打工人"形象（西装 + 领带 + 头顶时钟），屏幕中央显示嘲讽文字，直到答对一题才能解除
- 两种惩罚等概率随机触发

### 4.6 炸弹系统

**触发时机：** 爬楼过程中，随机从上方掉落炸弹

| 参数     | 值                                       |
| -------- | ---------------------------------------- |
| 出现概率 | 动态概率（楼层越高概率越低）             |
| 外观     | 黑色炸弹图标，带红色引信动画             |
| 掉落速度 | 快速下落，约 0.8s 穿过屏幕               |
| 碰撞判定 | 角色碰到炸弹 → 立即触发惩罚              |
| 预警特效 | 炸弹掉落前 0.5s 显示红色闪烁预警         |
| 躲避方式 | 左右滑动移动角色避开炸弹                 |

**炸弹动态概率机制：**
- 楼层 < 100：20%（0.2）
- 楼层 100-250：15%（0.15）
- 楼层 > 250：10%（0.1）

**炸弹惩罚机制（随机选择 A 或 B）：**

| 类型                 | 效果                                   | 触发流程                                                      |
| -------------------- | -------------------------------------- | ------------------------------------------------------------- |
| **惩罚 A：直接重来** | 游戏结束，从第 1 层重新开始            | 屏幕显示"被炸飞了！" → 3秒后回到开始界面                      |
| **惩罚 B：分享免死** | 弹出分享弹窗，分享给朋友可免除本次炸弹 | 弹窗显示"被炸飞了！分享给朋友可免除惩罚" → 用户选择分享或重来 |

**分享免死弹窗设计：**

- 标题："💥 被炸飞了！"
- 内容："分享给朋友，免除这次炸弹，继续爬楼！"
- 按钮：
  - [分享给朋友] → 触发微信分享 → 分享成功后继续游戏
  - [重新开始] → 从第 1 层重新开始
- 倒计时：10秒内未选择，默认重新开始

**分享失败保底机制：**
- 玩家累计分享失败 3 次后，自动解锁"免死"功能
- 第 3 次失败时显示："感谢你的尝试！已为你自动免除本次惩罚"
- 保底计数器存储在本地，避免玩家因分享失败而流失

**分享文案模板：**

- "我在爬楼游戏中被炸飞了！快来帮我免死吧！"
- "救救孩子！我爬到{当前层数}层被炸弹炸了..."
- "分享救我！继续逃离班味！"

### 4.7 礼包系统

**触发时机：** 放置在楼梯上，角色触碰即收集

| 参数     | 值                                       |
| -------- | ---------------------------------------- |
| 出现概率 | 动态概率（随通关次数增加）               |
| 外观     | 金色礼包盒，带闪光特效                   |
| 放置位置 | 随机放置在楼梯上                         |
| 碰撞判定 | 角色碰到礼包 → 立即触发奖励              |

**礼包动态概率机制：**
- 第一次通关：10%（0.1）
- 第二次通关：15%（0.15）
- 第三次通关：20%（0.2）

**礼包品质分级：**

| 品质 | 外观颜色 | 奖励内容 | 出现概率 |
|------|---------|---------|---------|
| **铜礼包** | 铜色 | 鼓励特效（文字 + 粒子） | 50% |
| **银礼包** | 银色 | 跳跃 +5 层 | 35% |
| **金礼包** | 金色 | 跳跃 +10 层 | 15% |

**鼓励文字池（随机一条）：**

- 加油！继续爬！
- 太棒了！
- 去班味成功！
- 你是最棒的！
- 坚持就是胜利！
- 马上就到顶了！
- 摸鱼达人！
- 逃离成功！

**奖励反馈：**

- 礼包碰撞后立即生效，无需额外操作
- 奖励 A 不影响游戏进度，仅提供正向反馈
- 奖励 B 直接推进游戏进度，增加爽感

### 4.8 终点与终局

| 参数         | 值                                           |
| ------------ | -------------------------------------------- |
| 终点楼层     | 第 365 层（"一年365天班"梗）                 |
| 到达终局触发 | 屏幕庆祝动画 → 直升机降落接人 → 撒大礼包祝福 |

**终局祝福语池（随机一条）：**

- 祝你从此无班味！天天摸鱼，天天加薪！
- 365天班？不存在的！你已成功裸辞！
- 直升机接走，从此只上自己想上的班！
- 班味已清空，满血复活！

**分享按钮：** 终局展示后出现"分享到朋友圈"按钮，附带楼层数和祝福语截图。

## 五、视觉与美术规格

### 5.1 整体风格

- **风格：** 简约卡通 Emoji 风
- **特点：** 干净扁平，操作一目了然，截图分享好看
- **角色：** Emoji 人物（👨‍💻 程序员 / 💼 打工人等）

### 5.2 色彩规格（暖色调）

| 用途         | 颜色                                    |
| ------------ | --------------------------------------- |
| 主色调       | #FFD84D（黄色，温暖 + 正向）            |

### 5.4 动效

| 动效      | 规格                                          |
| --------- | --------------------------------------------- |
| 角色移动  | 沿楼梯自动移动，带轻微弹跳                    |
| 角色跳跃  | 拉伸压缩动画，约 0.3s                         |
| 楼梯滚动  | Z型连续滚动，带拐角过渡                       |
| 障碍浮动  | 轻微上下浮动，0.8s 周期                       |
| 炸弹掉落  | 快速下落，约 0.8s 穿过屏幕，引信闪烁          |

| 礼包浮动  | 放置在楼梯上，轻微浮动，0.8s 周期             |
| 礼包开启  | 金色闪光 + 粒子爆发，持续 0.6s                |
| 鼓励特效  | 文字放大 + 彩色粒子飘落，持续 1s              |
| 跳跃+10层 | 角色快速向上跳跃 + "+10层"文字放大，持续 0.8s |
| 答对反馈  | 选项绿光 + 星星粒子，持续 0.5s                |
| 答错反馈  | 屏幕轻微震动（2px，0.3s）                     |
| 班味嘲讽  | 文字从中央放大出现，持续 0.4s，然后淡出       |
| 掉楼动画  | 快速下落，持续 0.4s                           |
| 终局庆祝  | 粒子飘落 + 直升机飞走，持续 3s                |

## 六、界面流程

```
启动闪屏
    ↓
开始界面
  ├─ [开始爬楼] → 进入游戏
  └─ [游戏说明] → 弹出简短说明（1屏）
    ↓
游戏主界面
  ├─ 楼层显示（左上角胶囊）
  ├─ 收集入口（右上角，点击进入收集展示）
  ├─ 暂停按钮（右上角）
  ├─ 角色（中央，沿Z型楼梯自动向上移动）
  ├─ Z型楼梯（连续滚动）
  ├─ 障碍物（❓ 方块，放置在楼梯上）
  ├─ 炸弹（💣，从上方掉落）
  └─ 礼包（🎁，放置在楼梯上）
    ↓ 点击收集入口
收集展示弹窗
  ├─ 摸鱼徽章格（已解锁/问号）
  ├─ 离职宣言格（已解锁/问号）
  ├─ 旅游目的地格（已解锁/问号）
  └─ [关闭]按钮
    ↓ 碰撞障碍
答题面板（弹出遮罩）
  ├─ 题目
  ├─ 选项列表
  └─ [确认]（或自动提交）
    ↓ 答题结束
反馈动效（0.5s）
    ↓
继续游戏 或 惩罚动画
    ↓ 碰撞炸弹
炸弹惩罚弹窗（弹出遮罩）
  ├─ 标题："💥 被炸飞了！"
  ├─ 内容："分享给朋友，免除这次炸弹，继续爬楼！"
  ├─ [分享给朋友] → 触发微信分享 → 分享成功后继续游戏
  └─ [重新开始] → 从第 1 层重新开始
    ↓ 碰撞礼包
礼包奖励（立即触发）
  ├─ 鼓励特效（文字 + 粒子）或
  └─ 跳跃 +5/+10 层（楼层增加，角色继续爬楼）
    ↓ 到达 365 层
终局界面
  ├─ 直升机动画
  ├─ 祝福语
  ├─ [再玩一次]
  └─ [分享到朋友圈]
```

## 九、风险与假设

| 风险/假设    | 说明                                                  | 解决方案 |
| ------------ | ----------------------------------------------------- | -------- |
| 题库规模     | 多轮游戏模式下，题库是否足够支撑不重复抽取            | ✅ 已扩充到100道 |
| 障碍频率     | 每 10-15 层为估计值，需实际测试调整                   | 待测试验证 |
| 惩罚力度     | 3-5 层掉落在测试中验证是否造成挫败感                  | ✅ 已采用分段惩罚+安慰文字 |
| 微信分享限制 | 祝福语截图传播依赖微信分享能力，需遵守分享规范        | 需遵守微信分享规范 |
| 炸弹概率     | 动态概率需测试验证是否过于频繁，影响游戏体验          | ✅ 已采用动态概率+预警特效 |
| 礼包概率     | 动态概率需测试验证是否足够激励，又不破坏平衡          | ✅ 已采用动态概率+品质分级 |
| 分享免死     | 微信分享成功率需测试，避免玩家因分享失败而流失        | ✅ 已采用3次失败保底机制 |
| 收集系统     | 三轮收集是否足够吸引玩家重复游玩，需测试验证          | ✅ 已添加实时进度展示 |
| 难度变化     | 楼梯速度递增是否影响游戏体验，需测试调整              | ✅ 已采用难度封顶+道具平衡 |
| 跳跃操作     | 长按跳跃的学习成本是否过高                            | 待测试验证，可考虑增加引导 |
