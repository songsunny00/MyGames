# 爬楼 (Going Up)

> 一款以"打工人逃离职场"为情感核心的跑酷答题微信小游戏

---

## 游戏简介

控制角色在 Z 形楼梯上不断向上跳跃，每隔 10~15 层遭遇一个 ❓ 问号方块触发答题。答对"去班味"答案继续爬，答错触发惩罚。最终爬到第 **365 层**顶楼，直升机接走撒祝福，完成一次精神离职。

---

## 玩法说明

| 操作 | 效果 |
|------|------|
| 左滑 / 右滑 | 角色左右移动 |
| 角色自动跳跃 | 无需额外操作 |

### 随机事件

- **❓ 问号方块**：每 10~15 层出现一次，碰触后进入答题模式
  - 答对 → 去班味成功，继续爬楼
  - 答错 → 随机触发**掉楼**（退 3~5 层）或**班味惩罚**（需再答对一题解除）
- **💣 炸弹**：随机从上方掉落，碰到后触发惩罚
  - 随机选择**直接重来**或**分享免死**弹窗（10 秒倒计时）
  - 分享失败累计 3 次后自动保底免除
- **🎁 礼包**：随机从上方掉落，碰到后立即奖励
  - 铜礼包：鼓励特效
  - 银礼包：+5 层
  - 金礼包：+10 层

### 目标

爬到第 **365 层**，触发终局动画（直升机 + 祝福语），实现"365 天逃离班味"。

## 项目结构

```
game03goingUp/
├── game.js                     # 微信小游戏入口
├── game.json                   # 小游戏配置（竖屏）
├── project.config.json         # 微信开发者工具配置
├── assets/
│   └── questions.json          # 题库（40 道，可扩充至 100 道）
└── js/
    ├── config.js               # 全局常量（物理参数、颜色、概率等）
    ├── main.js                 # 顶层状态机（START / PLAYING / END）
    ├── scenes/
    │   ├── StartScene.js       # 开始界面（含游戏说明）
    │   ├── GameScene.js        # 游戏主场景
    │   ├── QuizPanel.js        # 答题面板（底部滑入）
    │   ├── BombModal.js        # 炸弹惩罚弹窗
    │   └── EndScene.js         # 终局界面（直升机动画）
    ├── entities/
    │   ├── Character.js        # 角色实体（物理 + 绘制）
    │   ├── PlatformManager.js  # Z 形平台生成 / 滚动 / 回收
    │   └── ObstacleManager.js  # 炸弹 / 礼包掉落管理
    ├── systems/
    │   ├── InputSystem.js      # 触摸输入（wx / 浏览器双兼容）
    │   ├── QuizSystem.js       # 题库管理（局内不重复抽题）
    │   └── PunishmentSystem.js # 惩罚逻辑（掉楼 / 班味 / 分享保底）
    └── ui/
        ├── HUD.js              # 楼层显示 + 暂停按钮
        └── FeedbackFX.js       # 飘字 / 粒子 / 屏幕震动
```

## 开发与运行

### 在微信开发者工具中运行

1. 打开**微信开发者工具**，选择"小游戏"项目
2. 导入 `game03goingUp/` 目录
3. 将 `project.config.json` 中的 `appid` 替换为你的小游戏 AppID
4. 点击"编译"即可预览

### 本地 Node.js 快速验证

```bash
# 验证核心模块是否正常加载
node -e "const {GameScene}=require('./js/scenes/GameScene.js'); const gs=new GameScene(()=>{}); gs.init(0); console.log('floor='+gs.getCurrentFloor())"
```

## 题库扩充

题库位于 `assets/questions.json`，当前 40 道，设计目标 100 道。

每道题格式：

```json
{
  "id": 1,
  "type": "workplace",
  "question": "题目内容",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "answer": 1
}
```

`type` 可选值：`"workplace"`（职场反卷）或 `"brainteaser"`（脑筋急转弯）。`answer` 为正确选项的索引（0~3）。
