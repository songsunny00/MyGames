# 星空烟花沙盒 - UI/UX 设计规范

## 设计概述

| 维度 | 选择 | 理由 |
|------|------|------|
| **产品类型** | Entertainment / Casual Game | 解压沙盒游戏，主打沉浸式视觉体验 |
| **目标用户** | C端用户，全年龄段 | 简单交互，老少皆宜 |
| **设计风格** | Dreamy Watercolor (梦幻水彩) | 用户指定，与星空烟花主题完美契合 |
| **交互模式** | Touch + Motion | 触摸点火 + 重力感应控制 |

## 性能优化

### 渲染优化

```
┌─────────────────────────────────────────────────────────────┐
│  性能目标                                                    │
├─────────────────────────────────────────────────────────────┤
│  帧率        60 FPS (16.67ms/帧)                            │
│  粒子上限    500 个同时渲染                                  │
│  烟花上限    10 个同时上升                                   │
│                                                             │
│  优化策略:                                                  │
│  1. 粒子对象池 - 避免频繁 GC                                │
│  2. 离屏裁剪 - 不渲染不可见粒子                             │
│  3. 批量渲染 - 减少 Canvas 状态切换                         │
│  4. requestAnimationFrame - 平滑动画                        │
└─────────────────────────────────────────────────────────────┘
```

## 设计检查清单

### 视觉质量
- [x] 使用 SVG/矢量图标，无 emoji
- [x] 图标风格一致 (线性/填充统一)
- [x] 按下状态不改变布局边界
- [x] 语义化颜色标记

### 交互
- [x] 所有可点击元素提供按下反馈
- [x] 触摸目标 ≥ 44x44pt
- [x] 微交互动画 150-300ms
- [x] 禁用状态视觉清晰

### 布局
- [x] 尊安全区域 (状态栏/底部手势区)
- [x] 滚动内容不被固定栏遮挡
- [x] 8dp 间距节奏一致
- [x] 支持横竖屏

### 无障碍
- [x] 有意义的图标有 accessibility label
- [x] 颜色不是唯一指示器
- [x] 支持减少动画偏好
- [x] 状态变化可被屏幕阅读器识别
