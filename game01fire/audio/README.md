# 音频文件说明

本游戏需要以下音频文件，请将它们放置在 `audio/` 目录下：

## 所需文件

1. **bgm.mp3** - 背景音乐
   - 建议时长：2-3 分钟
   - 风格：舒缓、梦幻、星空氛围
   - 推荐来源：免费音乐网站如 Pixabay、Freesound

2. **launch.mp3** - 点火音效
   - 时长：0.5-1 秒
   - 效果：轻柔的"嗖"声，烟花升空的声音

3. **explode.mp3** - 爆炸音效
   - 时长：1-2 秒
   - 效果：烟花绽放的声音，柔和但有冲击感

## 免费音效资源推荐

- [Pixabay Music](https://pixabay.com/music/)
- [Freesound](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## 注意事项

- 音频文件格式必须是 MP3
- 文件名必须与上述完全一致
- 建议背景音乐文件大小控制在 500KB 以内
- 音效文件大小控制在 100KB 以内

## 临时测试

如果暂时没有音频文件，游戏仍然可以正常运行，只是没有声音效果。
